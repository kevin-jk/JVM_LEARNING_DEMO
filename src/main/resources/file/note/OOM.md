#OOM相关知识点

OOM就是out of memory.当相关的内存不足的时候，就会发生OOM. 

##可能发生OOM的区域

那么那些地方会发生OOM呢？首先回顾下java的内存布局。

这个知识点想必大家都是知道的，java的内存布局有：

1. 程序计数器
2. java栈
3. 本地方法栈
4. 方法区
5. 堆

根据我的理解，其实上面的5个内存区域都有可能发生OOM. 下面我们将对每一种情况进行分析。
